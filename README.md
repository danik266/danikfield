# DanikField

Сапёр, но не тот что ты помнишь.

## Что это

Веб-платформа для игры в сапёр. Четыре режима, три сложности, AI-коуч который подсказывает вероятности, ежедневные челленджи, скины, темы, рейтинг.

Собран на Vite + vanilla JS. Без фреймворков, без лишних зависимостей.

## Зачем

Существующие онлайн-сапёры выглядят как 2003 год. Хотелось сделать что-то, во что реально хочется играть — с нормальным дизайном, быстрым интерфейсом и фичами которые делают игру интереснее, а не просто повторяют оригинал.

## Фичи

- **Классика / Speed Run / Zen / Тренажёр** — разные способы играть
- **AI Coach** — анализирует поле, подсказывает безопасные ходы, учит думать вероятностями
- **Heatmap** — подсветка вероятности мин на каждой ячейке
- **Daily Challenge** — одна карта в день для всех, общий рейтинг
- **Магазин** — скины ячеек и темы оформления за XP или PRO
- **Stripe** — оплата PRO через Stripe Elements

## Запуск

```bash
npm install
npm run dev
```

Для Stripe — добавь ключ в `.env`:
```
VITE_STRIPE_PUBLISHABLE_KEY=pk_test_...
```

## Стек

Vite, JavaScript, CSS, Stripe.js, Lucide Icons
